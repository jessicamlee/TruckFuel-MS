Truck Fuel Modelling System

“Learn about fuel efficiency with our interactive education tool that helps kids visualize and optimize their fuel consumption.”

What is TFMS?

TFMS will be an education‐based fuel‐visualization tool for kids, that helps with calculating optimal fuel consumption while taking into account several mining factors throughout the haul cycle.

The TFMS tool provides value in two‐parts, first by demonstrating a sample visual model of fuel usage based on a given truck type and route, as well as providing calculations for the fuel usage for each of 5 truck types.
